### Approach

- Got the result via ternary operator and then printed it

### Code

```cpp
#include<iostream>

using namespace std;

int main()  {
    int x, y;
    cin>>x>>y;
    string result = (x>y) ? "Rahul" : "Robin";
    cout << result;
    return 0;
}
```
